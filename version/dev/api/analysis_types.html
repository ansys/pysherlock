Analysis Types#

Module containing types for the Analysis Service.

ComponentFailureMechanism

Contains the properties of a component failure mechanism update request.

PTHFatiguePropsAnalysis

Contains the properties of a PTH fatigue analysis update request.

SemiconductorWearoutAnalysis

Contains the properties of a semiconductor wearout analysis update request.

UpdateComponentFailureMechanismPropsRequest

Contains the properties of a component failure mechanism update per project.

UpdateLeadModelingPropsAnalysis

Contains the properties of lead modeling for an FEA analysis.

UpdateLeadModelingPropsRequest

Contains the properties of a lead modeling properties update request.

UpdateMechanicalPartsPropsAnalysis

Contains the properties of mechanical parts for an FEA analysis.

UpdateMechanicalPartsPropsRequest

Contains the properties of a mechanical parts properties update request.

UpdateMountPointsPropsAnalysis

Contains the properties of mount points for an FEA analysis.

UpdateMountPointsPropsRequest

Contains the properties of a mount points properties update request.

UpdatePTHFatiguePropsRequest

Contains the properties of a PTH fatigue analysis update per project.

UpdatePTHFatiguePropsRequestAnalysisType(*values)

Constants for qualification choices in the Update PTH Fatigue Properties request.

UpdateSemiconductorWearoutAnalysisPropsRequest

Contains the properties of a semiconductor wearout analysis update per project.

UpdateTraceModelingPropsAnalysis

Contains the properties of trace modeling for an FEA analysis.

UpdateTraceModelingPropsRequest

Contains the properties of a trace modeling properties update request.
